# 📈 Stock Market Analysis Dashboard

A professional fintech-inspired stock market analysis web application built using Python and Streamlit. The dashboard allows users to analyze stock market trends, visualize historical data, and explore real-time market insights through interactive charts and KPI cards.

---

# 🚀 Features

## ✅ Real-Time Stock Data

* Fetch live stock market data using Yahoo Finance API
* Display current stock prices and company information
* Support for multiple stock symbols

## ✅ Interactive Dashboard

* Modern dark gradient UI
* Responsive KPI cards
* Interactive sidebar controls
* Professional dashboard layout

## ✅ Data Visualization

* Interactive stock trend charts using Plotly
* Moving Average analysis
* Historical price tracking
* Dynamic trend visualization

## ✅ Error Handling

* Invalid stock symbol detection
* Empty API response handling
* Smooth user experience

# 📈 System Workflow

1. User enters a stock symbol
2. Application sends request to Yahoo Finance API
3. Data is retrieved and processed
4. KPIs and visualizations are generated
5. Results are displayed interactively
